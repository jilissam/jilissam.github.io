HTML Midterm Changelog!

by Jilissa Macias

Page 1 Errors

  • 6 HTML Errors
    1. Missing </li> ending tag: The third list item is missing its closing tag. To fix this you add the </li> tag.
    2. Missing </td> ending tag: The third cell in table 1 is missing its closing tag. To fix this you add </td> tag.
    3. Missing </table> ending tag: After the second table row, the </table> is missing, therefore you should add </table> to fix this.
    4. Missing </tr> ending tag: After the second table row, the </tr> is missing, therefore you should add </tr> to fix this.
    5. Missing </p> ending tag: After "This div contains a nested paragraph.", the closing paragraph tag is missing. To fix this you need to add the </p> tag.
    6. Image tag improperly closed: After the unordered list, the <img> tag is supposed to be closed using "/>" and not just ">".

Page 2 Errors

  • 3 HTML Errors
    1. The <b> tag is unordered: For the "Interior" heading, the <b> and </b> tag are outside of the <h2> tag. The b tag needs to be on the inside of h2.
    2. Two broken links: The "bay" and "segmental arch" links are both broken, because it is not a full absolute link. To complete the absolute link you need to add https:// in front of "www."
    3. Incomplete closing <b> tag: The "New Beehive Inn" in the first paragraph is wrapped in the b tag. However, the closing tag is "<>" and should be "</b>".
  • 1 Logic Error
    1. Visual Inconsistency: Overall, the page has hard readability, with inconsistent headers, font size, and unordered divider lines. Therefore, following our h1 header, I organized the sub headers to be h2. Additonally, I completed and reordered the divider lines to be consistent across the page.

Page 3 Errors

  • 2 HTML Errors
    1. Missing Title, Meta, and CSS Stylesheet Link: To fix this I like to use "! tab" to generate the meta charset, name, content, scale and title. I then added a link href to the cars css file and set rel to "stylesheet"
    2. Missing Doctype: It is important to always declare a doctype as the first line of code in a document. This page was missing the html doctype which prevented the browser from knowing how to interpret the page. So, I added it!
  • 1 Logic Error
    1. Mispositioned Footer: The footer that says "Generic Car Page. All rights reserved" was displaying on top of the form. It seemed to be written correctly in the html code, so I went to the css document and messed around with the footer style section. I changed the position from absolute to relative and that fixed the problem!

Page 4 Errors

  • 3 HTML Errors
    1. Missing Doctype: This page was missing the html doctype which prevented the browser from knowing how to interpret the page so, I added it.
    2. Missing <td> Tags: Some lines of code were missing their ending </td> tag or did not enclose their respective <img> tag, so I added them!
    3. Missing Alt Attributes Inside <img> Tag : Inside the <img> tag, the alt attribute was missing, which provides alternate text to an image. For accessibility purposes, it's best we add this.
  • 1 Logic Error

Instead of moving the images to be in the same column as the headers, I moved the headers to be in the correct rows by moving the <th> tags inside the <tr> tags.
